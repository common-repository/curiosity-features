== Description ==

Curiosity Features is a WordPress plugin for Curiosity Lite theme from Webzakt.
Get free or pro Curiosity WordPress Theme from webzakt.com.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/curiosity-features` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
